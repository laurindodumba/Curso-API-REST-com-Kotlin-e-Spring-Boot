#  Curso de REST API com Kotlin e Spring Boot

Este repositório faz parte do **Curso de Desenvolvimento de APIs REST utilizando Kotlin e Spring Boot**, com foco em boas práticas, arquitetura limpa e desenvolvimento profissional de back-end.

O curso aborda desde os **fundamentos** até a **construção de APIs REST completas**, incluindo segurança, persistência de dados e documentação.

---

##  Conteúdos Abordados

- Introdução ao Kotlin
- Fundamentos do Spring Boot
- Criação de APIs REST
- Controllers, Services e Repositories
- Spring Data JPA
- Integração com banco de dados
- Validações e tratamento de exceções
- DTOs e mapeamento de objetos
- Paginação e filtros
- Documentação com Swagger / OpenAPI
- Segurança com Spring Security (JWT)
- Boas práticas e arquitetura em camadas
- Testes automatizados (JUnit / Mockito)
- Deploy da aplicação

---

##  Tecnologias Utilizadas

- **Kotlin**
- **Spring Boot**
- Spring Web
- Spring Data JPA
- Spring Security
- Hibernate
- PostgreSQL / MySQL / H2
- Swagger / OpenAPI
- Maven ou Gradle
- JUnit 5
- IntelliJ IDEA
